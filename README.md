## Prerequisites📦 

- [A code Editor]
- [node globally installed]

### Cloning and Running the project 🛠️⚙

```
mkdir SatChat_Dapp
cd SatChat_Dapp
git clone https://github.com/JustARandomDude4/SatChat_Dapp.git

npm install gun

npm run build
npm run start

```
